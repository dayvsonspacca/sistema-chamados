# Sistema de chamados

Sistema basico de criação de chamados ou demandas, podendo vincular responsavel, niveis de prioridade, prazo e status do chamado.



## Instalação

Clone o projeto

```bash
  cd sistema-chamados

  composer install
  pnpm install

  php artisan migrate
  php artisan db:seed

```
